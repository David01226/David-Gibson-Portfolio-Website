TESLA

Project Overview

This project is a clone of the Tesla website. It draws on a range of functions such as intersection observers and DOM manipulation. The website is heavily video and image oriented and so this aspect in itself was one of the challenges faced whilst coding it, especially when it came to making it responsive. Likewise blending certain sections together with linear gradients and providing enough contrast between others was also a challenge.

RESPONSIVE LAYOUT

Through the use of media queries and breakpoints I have been able to make this website fully responsive for the user. It is compatible on mobile, tablet, small-medium desktop screens and even larger desktop screens.

Being heavily media based, it was important to ensure each video/image was clearly visible to the user throughout all screen dimensions. Sometimes this involved changing the flex-direction at certain break points or assigning 100% width for smaller screens.
