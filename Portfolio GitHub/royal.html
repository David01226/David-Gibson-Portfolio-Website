ROYAL LYTHAM GOLF CLUB

Project Overview

This is my most recent project, excluding this portfolio, and is a clone of the Royal Lytham golf club website. It combines a range of features which I have picked up over the past year such as, CSS grid, intersection observers and a few other neat little functional operations in javascript.

RESPONSIVE LAYOUT

Through the use of media queries and breakpoints I have been able to make this website fully responsive for the user. It is compatible on mobile, tablet, small-medium desktop screens and even larger desktop screens.
