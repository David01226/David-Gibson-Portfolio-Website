JYM ROOFING

Project Overview

JYM Roofing was one of the first full website clones I attempted to replicate. It allowed me to practice using jQuery to toggle on and off classes for the side menu. It was also the first time using splide.js for the image slider on the homepage. Other features include: on hover styling to reveal overlays, google maps via iFrame and flexbox positioning.
